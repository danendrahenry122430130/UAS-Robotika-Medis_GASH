Alur Kerja Sistem

  1. Input User ‚Üí Pengguna mengetik '1' (ON) atau '0' (OFF)
  2. Publish Data ‚Üí Node Publisher mem-publish ke topic /buzzer_cmd
  3. Network Transport ‚Üí ROS2 DDS middleware mengirim data
  4. Subscribe & Parse ‚Üí Node Subscriber menerima command
  5. Serial Transmission ‚Üí Data dikirim ke Arduino (9600 baud)
  6. Hardware Actuation ‚Üí Arduino aktifkan/nonaktifkan buzzer

Quick Start

Ikuti langkah berikut untuk menjalankan sistem:

1. Install ROS2 Jazzy

sudo apt install ros-jazzy-desktop -y
source /opt/ros/jazzy/setup.bash

2. Build Package

cd ~/ros2_ws
colcon build --packages-select buzzer_keyboard
source install/setup.bash

3. Run System

# Terminal 1
ros2 run buzzer_keyboard keyboard_publisher

# Terminal 2
ros2 run buzzer_keyboard buzzer_subscriber
